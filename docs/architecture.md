## 数据流摘要

| 阶段 | 触发源 | AI 处理 | 输出 | 人工卡点 |
|------|--------|---------|------|----------|
| A 灵感→PRD | Notion "Approved" | Claude 3.5 Sonnet | prd.json → GitHub | 无 (全自动) |
| B 编码→QA | prd.json push | Claude Code + Vision QA | 测试截图 + 报告 | 飞书 [确认发布] |
| C 上架→运营 | 飞书 [确认发布] | 自动回复/营销文案 | IPA → TestFlight | 飞书 [确认宣发] |

## 技术栈映射

| 组件 | 工具 | 部署位置 |
|------|------|----------|
| 编排引擎 | n8n (self-hosted) | Docker / Railway |
| CI/CD | GitHub Actions | GitHub |
| AI 编码 | Claude Code CLI | GitHub Runner |
| AI 视觉 | GPT-4o / Claude 3.5 Vision | API 调用 |
| E2E 测试 | Detox (RN) / Playwright | GitHub Runner |
| 打包发布 | Fastlane | GitHub Runner |
| 通信网关 | 飞书 Bot + Webhook | n8n 节点 |
| 数据源 | Notion API | 外部 SaaS |
| 监控 | Sentry + App Store Connect API | Cron (n8n) |
